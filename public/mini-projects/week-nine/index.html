Contact Form

Events used:

  1. Prevent Default
  2. Submit: When you input a name and click submit an alert appears.
  3. Input: When you type in the message box, it counts the character.
  4. Blur: When you type in an email with or without an @ and click out, a message appears.
  5. Mouse Over: When you hover over the button, it changes color.
  6. Mouse Out: When you move the mouse away from the button, it changes to pink.
  7. Focus: When the message box is in focus, it changes to purple
  8. Click: When you click the button another alert appears and the color changes.
  9. Keydown: If you press 'X', an alert appears.
  10. Keyup: Message appears when the X key is released
